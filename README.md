# AWA1 project

## install

- Clone the repo `git clone https://github.com/Westixy/AWA1.git`
- Go in the folder `cd AWA1`
- Install dependencies `npm i`
- Run app `npm start`
- Click on the button `Grow`

## file that are used

- `src/App.jsx` Entry point
- `src/btnGrow.jsx` what i did for the project
